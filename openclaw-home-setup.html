Key Findings on Frontier Models

671B Models Reality Check: DeepSeek V3.2 and Kimi K2.5 are technically "open weights" but require 240-380GB+ RAM even with aggressive quantization. These are NOT runnable on consumer Macs without significant compromises.
  • DeepSeek V3.2 (671B MoE): Requires ~380GB VRAM for Q4 quantization. Minimum viable: 2-bit quant at ~180GB with severe quality loss. Mac Studio M3 Ultra 512GB can run this at 10-20 tokens/sec.
  • Kimi K2.5 (1T MoE): Unsloth provides 1.8-bit dynamic quant at 240GB. With 256GB unified memory, expect ~10 tokens/sec. Full precision requires 4x H200 GPUs.
  • Qwen 2.5-Max: Realistic local target. M4 Pro 64GB runs 32B variant at 11-12 tokens/sec. 72B version needs Mac Studio.
  • Llama 3.3 70B: Sweet spot for local. Mac Studio M2 Ultra 64GB runs Q4 at usable speeds (~15-20 tokens/sec).

🎙️ Speech Recognition + TTS (Whole House)

Architecture Overview

A complete local voice assistant requires three components: Wake Word Detection (tiny, always-on), Speech-to-Text (Whisper), and Text-to-Speech (Piper/Coqui). For whole-house coverage, you need multiple satellite microphones with centralized processing.

🖥️ Updated Hardware Recommendations

Critical Question: Can Mac Mini 16GB Handle the Full Stack?

Answer: NO. A Mac Mini 16GB cannot simultaneously run:
  • Frontier LLM (even 8B model requires ~6GB)
  • Whisper small (~600MB + overhead)
  • TTS engine (~200-500MB)
  • Home Assistant / OpenClaw overhead

With macOS base usage, you'd have ~12GB available. This leaves no headroom for context windows, KV cache, or concurrent requests. The system would constantly swap, destroying latency.

🎯 Final Recommendation

BUY: Mac Studio M2 Ultra 64GB ($2,799)

Why this is the right choice:
  • Can run Llama 3.3 70B Q4 at usable speeds (~12-15 t/s)
  • Sufficient RAM for LLM + Whisper + TTS simultaneously
  • 800 GB/s memory bandwidth critical for large models
  • Break-even at ~28 months vs Claude Max 5x
  • Silent operation suitable for home server
  • Expandable to 128GB if needed (rare for current models)

DON'T BUY: Mac Mini 16GB ($599)

Why this won't work:
  • Cannot run 70B models (need 40GB+ RAM)
  • Limited to 8B models which are significantly less capable
  • No headroom for simultaneous Whisper + TTS
  • Would require separate hardware for voice stack
  • False economy - you'll upgrade within a year

Alternative: Mac Mini M4 Pro 64GB ($1,999)

If budget-constrained, this is the minimum viable option. It can run 70B models and the full voice stack, but with less headroom and slower inference than the Studio. Break-even improves to ~20 months.
